# XENIA DREAMS ✨

## 🌍 Descripción del proyecto

**XENIA DREAMS** es una plataforma web diseñada para dar oportunidades a estudiantes internacionales que desean mejorar su formación en España.

A través de un formulario online, los estudiantes pueden solicitar información y gestionar su proceso de admisión de forma sencilla, rápida y accesible.

---

## 🎯 Objetivo

Facilitar el acceso a la educación en España a estudiantes de todo el mundo, eliminando barreras administrativas y ofreciendo una experiencia digital moderna.

---

## 🚀 Funcionalidades principales

- 📝 Formulario de inscripción para estudiantes
- 📩 Envío de solicitudes de forma automática
- 🧑‍🎓 Gestión de datos de estudiantes
- 🌐 Interfaz moderna y responsive
- ⚡ Construido con **Tailwind CSS**

---

## 🛠️ Tecnologías utilizadas

- HTML5
- JavaScript
- PHP (backend)
- Tailwind CSS
- MySQL (base de datos)
